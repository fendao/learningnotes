# 6.1读写文本
**pd文件解析函数表**  
pd.read_csv('ex1.csv', header=None, skiprows=[0,2]  
        names=['a', 'b', 'c', 'message'], index_col=['key1', 'key2'],na_values={})  
        表头默认，跳行，创建表头，创建分层索引，指定缺失值标识  
pd.read_table('ex1.csv', sep='\s+')  
        作分隔符，  
**read_csv,read_table的参数表**
## 分块读入
读小部分行：pd.read_csv('ex1.csv', nrows=5)  
设置每块行数：pd.read_csv('ex1.csv', chunksize=1000) 并且可以遍历for piece in chunker
## 数据写入
逗号分隔文件：data.to_csv('out.csv')  
data.to_csv(sys.stdout, sep='|', na_rep='NULL', index=False, header=False, columns=['a','b'])  
            设置分隔符、标记缺失值、禁止行列标签，只要选定列
## 分隔格式
打开的文件对象传给csv.reader：reader = csv.reader(open('ex1.csv'))  
               遍历文件对象：for line in reader: print(line)  
             读成嵌套行列表：with open('ex1.csv') as f: lines = list(csv.reader(f))  
             读取列名和数据：header, values = lines[0], lines[1:]  
           列名数据配对字典：data_dict = {h:v for h,v in zip(header, zip(\*values))}  
直接定义自己方言类，或使用csv文件读取方言参数  
**csv方言参数表**  
csv.writer手动写入被分隔的文件
## json数据
json.loads(obj)上传成python  
json.dumps()装载成json  
pd.read_json('')将json按次序转成Series或DataFrame  
data.to_json(orient='records')导出为json
## XML HTML
import lxml, BeautifulSoup  
解析html：  
    pd.read_html('ex.html')搜索并解析table标签中的表格数据，返回DataFrame  
解析XML：  
    objectify.parse(open('ex.xml')).getroot()解析xml文件并获得根节点的引用  
    root.INDICATOR返回生成器，生成每一个indicator元素  
    objectify.parse(StringIO('<a href')).getroot()甚至能解析标签
# 6.2二进制
data.to_pickle('a')pickle格式写入硬盘  
pd.read_pickle('a')读取pickle  
pickle只能存几天
## HDF5分层数据格式
store = pd.HDFStore('ex.h5')创建文件路径与对象  
存储模式table：  
    store.put('obj', frame, format='table')将frame赋值给obj键，使用table模式  
    frame.to_hdf('ex.h5', 'obj', format='table')  
    pd.read_hdf('ex.h5', 'obj', where['index<5'])
## 读取excel
import xlrd, openpyxl  
文件对象：xlsx = pd.ExcelFile('ex1.xlsx')  
读为DataFrame：pd.read_excel(xlsx, 'Sheet1')直接文件名pd.read_excel('ex1.xlsx', 'Sheet1')  
写入excel：  
    生成EW再toexcel：frame.to_excel(pd.ExcelWriter('ex2.xlsx'), 'Sheet1')  
                或：frame.to_excel('ex2.xlsx')
# 6.3与WebAPI
import requests  
发送get：  
    url = 'baidu'  resp = requests.get(url)  
json解析为本地字典：  
    data = resp.json() 可直接生成DataFrame
# 6.4与数据库
import sqlite3  
建表语句、数据库连接语句、执行建表、执行插入数据语句、执行查找返回包含元祖的列表，  
            再利用查询结果的description属性的第一列，该列表可构造DataFrame  
            pd.DataFrame(rows, columns=[x[0] for x in cursor.description])  
import sqlalchemy as sqla  
pd.read_sql('select * from test', sqla.create_engine('sqlite:///mydata.sqlite'))
